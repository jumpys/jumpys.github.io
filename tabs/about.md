**Chirpy** is an responsive web design Jekyll theme developed using Bootstrap 4, designed by [Cotes Chung](https://github.com/cotes2020).

## Features

* Last modified date
* Table of Contents
* Disqus Comments
* Syntax highlighting
* Two Level Categories
* Search
* HTML compress
* Atom feeds
* Google Analytics
* Pageviews (Advanced)


## Used by

* [Cotes' Blog](https://blog.cotes.info)
